Instructions for setting up application database:
1. Install & set up SQL Server 2014.
2. Execute the script in .\db\TotaraPhoto.sql, plz note you may need to replace the physical path of the .mdf and .ldf file in the script, depending on the location you install your SQL Server 2014.
3. Update the db connection string in \TotaraPhotographyAssociation\Web.config according to your settings.
